  • 4. Write an Arrow function where it will do the following: a. It will take two array inputs b. Combine the two arrays and assign them in a new array c. Find the maximum number from the new array and return the result
  • 5. Write an Arrow function where it will do the following: a. It will take an array where the array element is even, push elements with even length to a new array and return the result. Print the result
